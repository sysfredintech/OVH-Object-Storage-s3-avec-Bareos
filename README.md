# OVH Object Storage S3 avec Bareos

## Ajout d'un stockage de type OVH Object Storage (s3) sur un serveur Bareos

 **Cette [documentation](./ajout-stockage-s3-ovh-bareos.md) fournit une méthode pour ajouter un stockage de type "Object Sorage" OVH sur un serveur Bareos.**

### Objectifs

- Ajouter un stockage cloud OVH Object Storage comme destination de sauvegarde
- Garantir le chiffrement des données (en transit et au repos)
- Optimiser les performances de transfert
- Valider la chaîne de sauvegarde par un test de restauration
